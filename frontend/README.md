# Building the frontend
To run a development version of the front end you must perform the following steps

In a terminal window:

1. cd ../jsc
2. yarn
4. yarn typechain
5. yarn dev

In a separate terminal window:

1. cd ../frontend
2. yarn
3. yarn update-all
4. yarn dev
